### 题1: 找出唯一成对的数

![](../images/wys1.png)

解题思路：

n^1^1 = n, 一个数两次^同一个数，得到数本身。

n^0 = n, 任何数异或0都是他自己。

将res^数组下标^数组的value，最终得出的结果就为重复值。

```python
def find_repeat(nums):
	res = 0
	for i in range(len(nums)):
		res ^= i
		res ^= nums[i]
	return res


print(find_repeat([1, 2, 3, 3, 4, 5, 6]))
```
